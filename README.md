# Установка и запуск проекта

### Шаги для локальной установки

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/1emax/test_rask.git kvach_test
   cd kvach_test
   ```
2. Запустите в терминале команды:
   ```bash
   cp .env.example .env
   docker-compose up -d
   ```

Проект будет доступен по адресу:
http://localhost:8000/
